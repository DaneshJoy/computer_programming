
# 📘 جزوه خلاصه Markdown

## 🧠 Markdown چیه؟

Markdown یه زبان ساده برای فرمت دادن متن هست که بدون نرم‌افزار خاص هم کار می‌کنه و خروجی تمیز (HTML / PDF / README) میده.

---

## 💻 روش استفاده

### 🔹 ویندوز / مک / لینوکس

* ساده‌ترین راه: فایل با پسوند `.md` بساز
* با اینا باز کن:

  * Notepad / VS Code / Typora
* برای پیش‌نمایش:

  * VS Code → `Ctrl+Shift+V`

### 🌐 آنلاین

* [StackEdit](https://stackedit.io/)
* [Dillinger](https://dillinger.io/)
* GitHub (برای README)

---

# ✍️ سینتکس پایه

## 🔹 تیترها (Headers)

```markdown
# H1
## H2
### H3
#### H4
##### H5
```

---

## 🔹 بولد و ایتالیک

```markdown
**متن بولد**
*متن ایتالیک*
***بولد و ایتالیک***
```

---

## 🔹 لیست‌ها

### بدون شماره

```markdown
- مورد اول
- مورد دوم
  - زیرمورد
```

### با شماره

```markdown

1. Item 1
1. Item 2
1. Item 3
    1. Item 3a
    1. Item 3b
        1. Item 3b1
        1. Item 3b2
```

- [ ] مورد اول
- [x] مورد دوم
  - [ ] زیرمورد
---

## 🔹 لینک و تصویر

### لینک

```markdown
[متن لینک](https://example.com)
```

### تصویر

```markdown
![متن جایگزین](image.jpg)
```
